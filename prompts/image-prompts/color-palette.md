# Paleta de Cores — Sabedoria de Valinor

## Cores Primárias

| Nome | Hex | Uso |
|------|-----|-----|
| Obsidiana (fundo) | `#0D1117` | Fundo principal de vídeos e infográficos |
| Ouro de Valinor | `#C9A84C` | Títulos, destaques, bordas, linhas |
| Azul de Mandos | `#1E3A5F` | Acentos, backgrounds secundários, gradientes |
| Branco de Taniquetil | `#F0E6D3` | Texto principal |
| Prata de Varda | `#A8B2C1` | Texto secundário, legendas |

## Gradientes

| Nome | Combinação | Uso |
|------|-----------|-----|
| Crepúsculo Élfico | `#0D1117` → `#1E3A5F` | Fundo de vídeos |
| Amanhecer de Valinor | `#C9A84C` → `#F0E6D3` | Destaques, títulos |
| Sombra de Mordor | `#0D1117` → `#2C1810` | Temas sombrios |
| Luz de Eärendil | `#1E3A5F` → `#C9A84C` | Cenas mágicas |

## Uso por Tipo de Conteúdo

| Conteúdo | Cores Dominantes |
|----------|-----------------|
| Lore épico / batalhas | Obsidiana + Ouro + Vermelho (#8B0000) |
| Elfos / Valinor | Obsidiana + Ouro + Azul de Mandos |
| Hobbits / Shire | Obsidiana + Verde (#2D5A27) + Ouro |
| Trevas / Mordor | Obsidiana + Vermelho (#8B0000) + Cinza (#3C3C3C) |
| Mistério / Tom Bombadil | Obsidiana + Roxo (#4A1A6B) + Ouro |

## Infográficos

- Fundo: sempre Obsidiana (#0D1117)
- Títulos: Ouro de Valinor (#C9A84C)
- Texto: Branco de Taniquetil (#F0E6D3)
- Linhas/conexões: Prata de Varda (#A8B2C1)
- Acentos: Azul de Mandos (#1E3A5F)
