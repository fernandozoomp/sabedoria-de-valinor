# Template de Prompt de Imagem

## Estrutura Base

Todo prompt de cena deve seguir esta estrutura:

```
[DESCRIÇÃO DA CENA], [ESTILO ARTÍSTICO], [ILUMINAÇÃO], [COMPOSIÇÃO], [FORMATO]
```

## Componentes

### Descrição da Cena
- Seja específico e detalhado
- Inclua: personagens, cenário, ação, atmosfera
- Ex: "Ancient Ent standing in a dark forest, glowing yellow eyes, moss covering body, arms raised"

### Estilo Artístico (obrigatório)
Use UMA destas opções:
- `cinematic concept art, Peter Jackson LOTR movie aesthetic`
- `dark fantasy oil painting, Alan Lee illustration style`
- `epic matte painting, John Howe style`
- `photorealistic fantasy landscape, Thomas Kinkade lighting`
- `medieval manuscript illustration, illuminated page style` (pra infográficos/lore)

### Iluminação
- `dramatic golden hour lighting`
- `moonlight through clouds, silver-blue tones`
- `volcanic red glow, Mount Doom aesthetic`
- `soft ethereal light, Lothlórien glow`
- `stormy battlefield, lightning flashes`

### Composição
- `wide establishing shot`
- `close-up portrait, shallow depth of field`
- `bird's eye view, sweeping landscape`
- `low angle, heroic perspective`
- `symmetrical composition, centered`

### Formato (obrigatório no final)
- `9:16 vertical format, 1080x1920, optimized for TikTok`

## Exemplo Completo

```
Ancient Ent standing in a dark forest, glowing yellow eyes, moss and bark 
covering massive body, arms raised in anger, smaller trees cowering below, 
cinematic concept art, Peter Jackson LOTR movie aesthetic, dramatic golden 
hour lighting with storm clouds, wide establishing shot, 9:16 vertical 
format, 1080x1920, optimized for TikTok
```

| Paisagem | `[location name], vast landscape, [terrain features], atmospheric perspective...` |
| Mapa | `antique fantasy map of [region], parchment texture, ink and watercolor illustration, compass rose...` |
