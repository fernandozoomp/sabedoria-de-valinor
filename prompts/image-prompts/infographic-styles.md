# Estilos de Infográfico

## Estilos Disponíveis

### 1. Mapa Antigo (cartography)
- Fundo: pergaminho envelhecido com textura
- Bordas: decoradas com motivos élficos
- Cores: sépia (#D4A574), marrom escuro (#5C3D2E), tinta azul (#2C5F7C)
- Fonte: estilo caligrafia medieval
- Elementos: bússola, rosa dos ventos, criaturas marinhas
- Uso ideal: geografia, rotas de viagem, localizações de batalhas

### 2. Linha do Tempo (chronicle)
- Fundo: pergaminho escuro (#0D1117) com bordas douradas (#C9A84C)
- Eixo vertical ou horizontal com marcadores
- Cada evento com ícone + data + descrição curta
- Imagens pequenas nos marcadores (se disponível)
- Uso ideal: história das Eras, vida de personagens, eventos cronológicos

### 3. Genealogia (family tree)
- Fundo: escuro (#0D1117)
- Linhas douradas (#C9A84C) conectando nomes
- Caixas com nome + título + anos de vida
- Ícones: coroa (reis), estrela (elfos), montanha (anões), folha (hobbits)
- Uso ideal: linhagens, heranças, relações entre personagens

### 4. Tabela Comparativa (comparison)
- Layout: duas ou mais colunas lado a lado
- Fundo escuro, cabeçalho dourado
- Ícones representando cada item comparado
- Checkmarks/xes pra sim/não
- Uso ideal: livro vs. filme, personagens comparados, raças comparadas

### 5. Mapa Mental (knowledge web)
- Centro: tema principal em destaque
- Ramos: subtópicos conectados por linhas
- Cores diferentes por categoria
- Fundo escuro com brilho sutil nos nós
- Uso ideal: explicações complexas, conexões entre eventos

## Regras Gerais

- Sempre 1080x1920 (vertical)
- Texto legível (mínimo 24px equivalente)
- Não sobrecarregar — máximo 6-7 elementos por infográfico
- Paleta fixa do projeto (ver `color-palette.md`)
- Sempre incluir o logo/brand no canto inferior
